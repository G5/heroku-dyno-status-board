# Heroku Dyno Status Board

Users will be able to log in and manage their Heroku app dynos.

A work in progress.


## Setup

Install required gems

```bash
bundle
```

Start the server

```bash
./start.sh
```

This app requires a self-signed SSL cert to run locally.
